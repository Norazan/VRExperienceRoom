◆ ScanForArduinos()

void VRExperienceRoom.IOHandler.ScanForArduinos ( )

Loops through all available COM ports to check if an Arduino running the VR Experience Room software is available. All comforming COM ports are added to the ports list.

The method checks for an Arduino by writing an '@' character to a COM port. If it receives an '@' character back, that means there is a Arduino attached.

Writes the given settings to a COM port. Should only be called using the Form.SendSettings(SerialPort, string, bool, bool) method!

The following communication protocol is used by the Arduino to interpret a setting through serial communication.

Wind RPM (0 - 5200) + '\n' + Heat Off/On (0/1) + '\n' + Scent Off/On (0/1) + '\n'

The following example serial string will turn the fans on to 1500 RPM, turn the heater On and scent Off.

1500\n1\n0\n
Parameters
portThe COM port to send the settings through.
WindRPMThe RPM setting for the fans.
heatThe on or off setting for the heater.
scentThe on or off setting for the mist maker.
